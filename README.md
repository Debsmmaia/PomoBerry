# 🍓 PomoBerry - Pomodoro Timer

O PomoBerry é um cronômetro Pomodoro interativo desenvolvido como um projeto prático de estudo focado nos fundamentos do ecossistema React. A aplicação tem como objetivo auxiliar na gestão de tempo e na produtividade, alternando automaticamente entre ciclos de foco total (25 minutos) e períodos de relaxamento (5 minutos). Para tornar a experiência mais imersiva e visualmente agradável, a interface reage dinamicamente ao tempo restante. Ela utiliza a biblioteca react-liquid-gauge para exibir um medidor central em formato de onda líquida animada e altera suavemente a cor de fundo de toda a página para refletir o estado atual do usuário.
